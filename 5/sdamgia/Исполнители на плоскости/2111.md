#### Решение #1 (Python)

```python
x = 0
y = 0

# Сместиться на вектор (5, 2)
x += 5
y += 2

# Сместиться на вектор (-3, 3)
x += -3
y += 3

# Повторить 3 [Сместиться на вектор (1, 0)]

for i in range(3):
	x += 1
	y += 0

# Сместиться на вектор (3, 1)
x += 3
y += 1

# Расстояние
d = (x ** 2 + y ** 2) ** 0.5

print(d)
```
**Ответ:** 10.
