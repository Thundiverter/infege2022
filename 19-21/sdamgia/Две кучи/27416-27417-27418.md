#### Решение #1 (Python)
```python
# x       количество камней, данное по условию
# S       значение, которое нам надо найти
# turn    счётчик ходов (чётное - Петя, нечётное - Ваня)

# Дано:
# x = 7 (камней)
# Smax = 77 (камней)
Smax = 77
x = 7

# 1 <= S <= 69

# Действия:
# 1) + 1;
# 2) x 2;

# Найти: S

def f(x, S, turn):
    global Smax
    
    if turn == 2:
        return x + S >= Smax
    else:
        return f(x + 1, S, turn + 1) or f(x, S + 1, turn + 1) or f(x * 2, S, turn + 1) or f(x, S * 2, turn + 1)

for i in range(1, 69):
    if f(x, i, 0):
        print(i)
        break
```
**Ответ:** 18.
