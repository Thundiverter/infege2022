#### Решение #1 (Python)
```python
from itertools import *

a = product('0123456789', repeat=2)

for i in a:
	s = f"12345{i[0]}6{i[1]}8"
	
	if int(s) % 17 == 0:
		print(s, int(s) // 17)
```
