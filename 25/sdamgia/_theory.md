#### Решение заданий 25
##### Тип 1
Найти числа, имеющие определённое кол-во простых делителей, и эти делители вывести.

```python
for i in range(<начало отрезка>, <конец отрезка> + 1):
	dels = []
	
	for j in range(2, i // 2 + 1):
		if i % j == 0:
			dels.append(j)

	if len(dels) == <требуемое кол-во делителей у числа>:
		print(dels)
```
