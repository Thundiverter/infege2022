#### Значение выражения 216^5 + 6^3 − 1 − x записали в системе счисления с основанием 6, при этом в записи оказалось 12 цифр 5. При каком минимальном целом положительном x это возможно?
```python
for x in range(10000):
    y = 216**5 + 6**3 - 1 - x
    
    y6 = ''
    while y > 0:
        y6 = str(y % 6) + y6
        y //= 6
    
    if y6.count('5') == 12:
        print (x)
        break
```
**Ответ:** 259.
