> Значение выражения 64^108 - 64^104 + 8^56 - 8^17 - 47 (^ означает степень) записали в системе счисления с основанием 7. Сколько получилось там цифр 0?

#### Решение #1 (Python)
```python
n = 64**108 - 64**104 + 8**56 - 8**17 - 47
s = ''

while n > 0:
    s = str(n % 7) + s
    n //= 7

print( s.count('0') )
```

**Ответ:** 38.
