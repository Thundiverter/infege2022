#### Условие:

> Решите уравнение
>   121<sub>x</sub> + 1 = 101<sub>7</sub>
> Ответ запишите в троичной системе счисления. Основание системы счисления указывать не нужно.

#### Решение #1 (Python):
```python
for i in range(3, 16):
    s = ''
    if int('121', i) + 1 == int('101', 7):
        while i > 0:
            s = str(i % 3) + s
            i //= 3
        print(s)
```
