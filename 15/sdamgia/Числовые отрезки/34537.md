#### Решение #1 (Python)
```python
def f(x, A):
	return ((x in A) <= (10 <= x <= 15)) == ((10 <= x <= 20) <= (5 <= x <= 15))

A = []

for i in range(1, 10000):
	i = i / 10
	if not f(i, A):
		A.append(i)

print(A)
```
Получаем значения от ``15.1`` до ``20.0``. Следовательно, x ∈ (15; 20]. Значит, ``20 - 15 = 5``.

**Ответ:** 5.
