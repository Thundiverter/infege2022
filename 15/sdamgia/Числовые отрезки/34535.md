#### Решение #1 (Python)
```python
def f(x, A):
	return ((x in A) or (10 <= x <= 40)) or ((5 <= x <= 15) <= (35 <= x <= 50))

A = []

for i in range(1, 10000):
	i = i / 10
	if not f(i, A):
		A.append(i)

print(A)
```
Получим список чисел от ``5.0`` до ``9.9`` без прерываний. Следовательно, x ∈ [5; 10). Значит, ``10 - 5 = 5``.

**Ответ:** 5.
