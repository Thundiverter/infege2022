#### Решение #1 (Python)
```python
minlen = 100000

P = range(10, 40 + 1)
Q = range(5, 15 + 1)
R = range(35, 50 + 1)

for a1 in range(100):
    for a2 in range(100):
        flag = True
        
        for x in range(-100, 100):
            if ((a1 <= x <= a2) or (x in P)) or ((x in Q) <= (x in R)):
                pass
            else:
                flag = False
                break
        
        if flag:
            minlen = min(minlen, a2 - a1)

print(minlen + 1) # т.к. промежуток
```
**Ответ:** 5.
