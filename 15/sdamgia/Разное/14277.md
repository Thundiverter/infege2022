#### Решение #1 (Python)
```python
def f(x, A):
    return (x not in A) <= (((17 <= x <= 40) and (20 <= x <= 57)) <= (x in A))

A = []
for i in range(1, 1000):
    i /= 10
    if not f(i, A):
        A.append(i)

print(A)
```

Получим список чисел от ``20.0`` до ``40.0`` без прерываний и полуинтервалов. Значит, ``40 - 20 = 20``.

**Ответ:** 20.
