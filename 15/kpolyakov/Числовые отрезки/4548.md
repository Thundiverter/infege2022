#### Решение #1 (Python)
```python
def f(x, a):
    return (x in a) <= ((not (26 <= x <= 52)) == (14 <= x <= 44))

A = [i/10 for i in range(1000)]

for i in range(1000):
    i /= 10
    if not f(i, A):
        A.remove(i)

print(A)
```

Получим список чисел от ``14.0`` до ``25.9`` и от ``44.1`` до ``52.0``. Получается, x ∈ [14, 26) U (44, 52]. Найдём длину каждого из промежутков:
* 26 - 14 = 12;
* 52 - 44 = 8.

Выбираем самый длинный.

**Ответ:** 12.
