#### Решение #1 (Python)
```python
from itertools import *

def f(x):
    P = 44 <= x <= 49
    Q = 28 <= x <= 53
    A = a1 <= x <= a2
    return (A <= P) or Q

oX = [i / 4 for i in range(0, 53*4+1)]
m = []

for a1, a2 in combinations(oX, 2):
    if all(f(x) for x in oX):
        m.append(a2-a1)

print(max(m))
```
**Ответ:** 25.

#### Решение #2 (Python)
```python
from itertools import *

def f(x):
    return ((a1 <= x <= a2) <= (44 <= x <= 49)) or (28 <= x <= 53)

oX = [i / 4 for i in range(0, 53*4+1)]
m = set()

for a1, a2 in combinations(oX, 2):
    flag = True
    for x in oX:
        if f(x) == False:
            flag = False
            break
    
    if flag:
        m.add(a2-a1)

print(max(m))
```
**Ответ:** 25.
