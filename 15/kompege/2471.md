#### Решение #1 (Python)
```python
def f(x, a):
	return (25 <= x <= 120) <= ( ( not(1 <= x <= 40) and (25 <= x <= 120) ) <= (x in a) )

A = []

for i in range(1500):
	i /= 10
	if not f(i, A):
		A.append(i)

print(A)
```

Получаем список чисел от ``40.1`` до ``120.0``. Значит, x принадлежит (40, 120]. Значит, 120 - 40 = 80.

**Ответ:** 80.
