```python
f = open('27-A.txt')
n = int(f.readline())

a = []
ma = 0
for i in range(n):
    a.append( int(f.readline()) )

for a0 in range(0, n):
    for a1 in range(a0, n):
        k = 0
        s = 0
        for x in range(a0, a1 + 1):
            s += a[x]
            if a[x] % 2 == 0:
                k += 1
        
        if k % 10 == 0:
            ma = max(ma, s)

print(ma)
```
**Ответ:** 4779554.
