#### Условие:

> Какое максимальное значение переменной s, подаваемого на вход программе, для которого в результате работы программы на экран будет выведено значение 46? 

#### Решение #1 (Python):
```python
arr = []
for i in range(1000):
    s = i
    n = 1
    while s > 200:
      s = s - 15
      n = n + 3
    if n == 46:
        arr.append(i)

print(max(arr))
```

#### Ответ: `425`
