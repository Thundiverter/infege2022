#### Решение #1 (Python)
Общая функция:
```python
# Нечётные ходы — Петя
# Чётные ходы — Ваня

def f(s, c, m):
	if s >= 177: return c%2 == m%2
	if c == m: return 0
	h = [f(s+1, c+1, m), f(s*2, c+1, m)]
	return any(h) if c%2 != m%2 else all(h)
```
