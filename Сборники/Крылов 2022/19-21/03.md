##### Задание 19
Из 100 Петя может получить 101 и 200, из которых Ваня может получить 201, 202 и 400 и т.о. выигрыать.

**Ответ:** 100.

##### Задание 20
```python
def f(s, t):
	if s >= 201 or t > 4:
		return t == 4
	if t % 2 != 0:
		return f(s+1, t+1) or f(s*2, t+1)
	return f(s+1, t+1) and f(s*2, t+1)

for i in range(1, 200+1):
	if f(i, 1):
		print(i)
```
