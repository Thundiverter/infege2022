##### Задание 19
Из 90 Петя может получить 91 и 180, а Ваня из них — 181, 182 и 360, тем самым выигрывая.

**Ответ:** 90.

##### Задание 20
```python
def f(x, t):
    if x >= 181 or t > 4:
        return t == 4
    if t % 2 != 0:
        return f(x+1, t+1) or f(x*2, t+1)
    return f(x+1, t+1) and f(x*2, t+1)

for i in range(1, 180 + 1):
    if f(i, 1):
        print(i)
```
