#### Решение #1 (Python)
```python
def f(x, A):
    B = 30 <= x <= 41
    C = 50 <= x <= 56
    return (not ( (B or C) <= (x in A) ))

A = []

for i in range(56*10+1):
    i /= 10
    if f(i, A):     # без not, т.к. выражение ложно
        A.append(i)

print(A)
```

Получим отрезок от ``30.0`` до ``56.0``. Значит, длина отрезка: 56 - 30 = 26.

**Ответ:** 26.
