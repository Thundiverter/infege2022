#### Решение #1 (Python)
```python
count = 0
n = 860000

while count < 5:
    n += 1
    dels = set()
    for j in range(2, int(n**0.5)+1):
        if n % j == 0:
            dels.add(j)
            dels.add(n//j)
    
    if len(dels) == 0: continue
    
    M = min(dels) + max(dels)
    
    if str(M)[-2:] == '18':
        print(n, M)
        count += 1
```
