#### Решение #1 (Python)
```python
from itertools import *

a = product('0123456789', repeat=2)

for i in a:
    x, y = i[0], i[1]
    
    n = int( '12345' + x + '6' + y + '8' )
    
    if n % 17 == 0:
        print(n, n // 17)
```
