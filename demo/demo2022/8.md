```python
# Е Л М Р У
# 0 1 2 3 4

# длина слов - 4
# мощность алфавита - 5

count = 0
r = range(5)

for a in r:
	for b in r:
		for c in r:
			for d in r:
				count += 1
				if a == 1:
					print(count)
```

Получаем порядковые номера всех слов, начинающихся с Л. Поскольку нужен порядковый номер первого слова, выбираем самое первое число.

**Ответ:** 126.
