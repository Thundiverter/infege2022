#### Условие:

> Ниже записаны две рекурсивные функции (процедуры): F и G. Сколько символов «звёздочка» будет напечатано на экране при выполнении вызова F(11)?

#### Решение #1 (Python):
```python
def F(n):
  if n > 0: G(n - 1)

def G(n):
  print("*")
  if n > 1: F(n - 3)

F(11)
```

#### Ответ: `3`
