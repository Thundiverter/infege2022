#### Пример:
> Дана строка **QRAGQT**. Нужно найти самое большое расстояние между одинаковыми элементами строки.

#### Решение #1 (Python):
```python
s = 'QRAGQT'
maxLen = 0
for el in s:
    o1 = s.index(el)
    o2 = len(s) - 1 - s[::-1].index(el)
    maxLen = max(maxLen, o2-o1)
print(maxLen) # >> 4
```
