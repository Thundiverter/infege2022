#### Условие:

> Найдите битовую глубину кодирования растрового изображения размером 512x300 пикселей, которое занимает 600 Кбайт. 
> В ответе запишите только целое число, единицу измерения писать не нужно.

#### Решение #1 (Python):
```python
s = 512 * 300
memory = 600 * 2**13
i = int(memory / s)
print(i)
```
