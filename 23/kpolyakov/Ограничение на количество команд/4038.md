#### Решение #1 (Python)

```python
a = set()

for x in range(2**11):  # (кол-во действий) ** (кол-во комманд)
    s = ''
    n = x
    for i in range(11):
        s= '12'[n%2] + s
        n //= 2
    
    v = 3
    for c in s:
        if c == '1':
            v += 1
        if c == '2':
            v = v*2+1
    
    a.add(v)

print(len(a))
```
