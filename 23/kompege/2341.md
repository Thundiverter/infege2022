#### Условие:
> Сколько разных чисел на отрезке [1000, 1024] может быть получено из числа 1 с помощью программ, состоящих из 8 команд?

#### Решение (Python)
```python
a = set()
def f(x, c):
    if c == 8 and 1000 <= x <= 1024:
        return a.add(x)
    if c < 8:
        f(x+1, c+1)
        f(x+5, c+1)
        f(x*3, c+1)
f(1, 0)
print(len(a))
```
