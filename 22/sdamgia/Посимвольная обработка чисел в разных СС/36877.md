#### Решение #1 (Python)
Нужно, чтобы *a* и *b* изначально не были отрицательными. Для этого начинаем перебор с ``6``, чтобы *b* не был отрицательным.
```python
for i in range(6, 1000):
    x = i
    a = 3*x + 23
    b = 3*x - 17
    while a != b:
        if a > b:
            a -= b
        else:
            b -= a
    
    if a == 10:
        print(i)
        break
```
**Ответ:** 9.
