#### Решение #1 (Python)
```python
f = [int(i.replace('\n', '')) for i in open('17_37336.txt', 'r').readlines()]

pairsCount = 0
maxSum = -10000000

for i in range(1, len(f)):
	if f[i] % 3 == 0 or f[i - 1] % 3 == 0:
		pairsCount += 1
		
		s = f[i] + f[i - 1]
		if s > maxSum:
			maxSum = s

print(pairsCount, maxSum)
```
**Ответ:** 2802 1990.
