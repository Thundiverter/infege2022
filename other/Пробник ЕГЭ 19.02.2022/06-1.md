> Определите, при каком **наименьшем** введённом значении переменной **s** программа выведет число 343. Для Вашего удобства программа представлена на четырёх языках программирования.

```python
# PYTHON
s = int(input())
n = 1
while s <= 70:
    s = s + 9
    n = n * 7
print(n)
```

#### Решение #1 (Python)
```python
for i in range(-1000, 10000):
    s = i
    n = 1
    while s <= 70:
        s = s + 9
        n = n * 7
    if n == 343:
        print(i)
        break
```
**Ответ:** 44.
