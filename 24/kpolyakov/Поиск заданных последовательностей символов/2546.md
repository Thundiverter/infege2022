#### Решение #1 (Python)
```python
import re
f = open('24-j5.txt')
s = f.readline().replace('KOT', 'X')

m = s.replace('S', ' ').replace('T', ' ').replace('O', ' ').replace('C', ' ').replace('K', ' ')
l = [len(i) for i in m.split()]

print(max(l))
```

#### Решение #2 (Python)
```python
f = open('24-j5.txt')
s = f.readline()

p = s.find('KOT')
count = 1
maxcount = 0
while p > -1:
    s = s[p+3:]
    p = s.find('KOT')

    if p == 0:
        count += 1
    else:
        maxcount = max(maxcount, count)
        count = 1

print(maxcount)
```
**Ответ:** 2.
